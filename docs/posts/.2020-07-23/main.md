# 문제점

1. 이미지 삽입이 안 됨
2. 코드 삽입하면 깨짐.

이런 문제들이 발생하는 원인은 딱 하나인데, 뭐냐면 HTML을 SPA react app에 import하기 위해 막무가내로 삽입했기 때문.

md파일을 jsx파일로 바꿀 수 있는 부드러운 방법이 필요하다고 생각됨.

markdown-it에서 여러 기능을 제공하는 것 같던데 그것도 생각중이고, 직접 컴파일러를 짤까도 고민중.
